# Bike Sharing Dashboard 🚲

**Python Version:** 3.9 atau lebih baru

## Setup Environment

Disarankan menggunakan virtual environment agar dependencies terisolasi dengan baik.

1. Buka terminal
2. Buat virtual environment baru:
   ```bash
   python -m venv venv

3. Aktifkan virtual environment:

- Windows: venv\Scripts\activate
- Mac/Linux: source venv/bin/activate

4. Install library yang dibutuhkan

pip install -r requirements.txt

## Run Streamlit App

Masuk ke direktori utama (atau direktori dashboard/ tempat file .py berada) Jalankan perintah berikut:

streamlit run dashboard.py
